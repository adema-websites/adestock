2. Aviso de Seguridad de Windows

¿Por qué Windows avisa que "no es seguro"?
El instalador y el ejecutable aún NO cuentan con firma digital. Por eso, SmartScreen o tu antivirus pueden mostrar una advertencia porque no reconocen el editor.
  • El software es seguro: solo contiene el intérprete de Python y los archivos necesarios para ejecutar ADEstock en tu equipo.
  • Para continuar, hacé clic en “Más información” → “Ejecutar de todas formas”. Windows recordará tu decisión y no volverá a preguntar.

3. Instalación Paso a Paso

  1. Descomprime el archivo adestock-demo.zip en una carpeta de tu PC (por ejemplo, en el Escritorio).
  2. Abre la carpeta descomprimida y ejecuta el archivo adestock-demo.exe (o el instalador correspondiente).
  3. Si Windows muestra una advertencia, sigue las instrucciones del paso anterior.
  4. El instalador creará la carpeta Documentos\ADEstock y un acceso directo en el Escritorio.
  5. No se requiere conexión a internet ni configuración avanzada.

GIF: Instalación paso a paso

4. Cómo Iniciar ADEstock

  1. Haz doble clic en el acceso directo ADEstock del Escritorio.
  2. Se abrirá una ventana negra (CMD). ¡No la cierres! Esa ventana es el servidor local y debe permanecer abierta mientras uses el sistema.
  3. En la consola verás dos enlaces, por ejemplo:
    http://127.0.0.1:8965 (solo este PC)
    http://192.168.0.123:8965 (cualquier dispositivo de tu red)
  4. Con el ratón dentro de la ventana, clic derecho → Abrir vínculo (o copia y pega en tu navegador).
  5. ADEstock se cargará en el navegador que prefieras (Chrome, Edge, Firefox, etc.). ¡Listo para trabajar, sin necesidad de internet!

GIF: Cómo acceder y usar la demo

5. Acceder desde otros dispositivos de la misma red

  • Asegúrate de que la PC anfitrión (donde instalaste ADEstock) esté encendida y con ADEstock ejecutándose.
  • Desde el móvil, tablet u otra PC conectada al wifi, ingresa la segunda dirección que aparece en la consola (la que contiene tu IP local).
  • Si no carga, verifica que ningún firewall bloquee el puerto 8965.

6. Cómo cerrar ADEstock

  • Cierra la ventana CMD para detener el servidor.
  • Para volver a usar ADEstock, solo haz doble clic en el acceso directo nuevamente.
