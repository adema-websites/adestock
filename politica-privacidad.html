Política de Privacidad

1. Introducción

En ADEstock, desarrollado por ADEMA Sistemas, valoramos y respetamos la privacidad de nuestros usuarios. Esta Política de Privacidad describe cómo recopilamos, utilizamos, almacenamos y protegemos la información personal que nos proporcionas al utilizar nuestro software y sitio web.

2. Información que Recopilamos

  • Datos de contacto: Nombre, correo electrónico, teléfono y empresa (cuando los proporcionas para solicitar una demo, soporte o adquirir el sistema).
  • Datos de uso: Información sobre cómo utilizas el software, solo con fines de mejora y soporte.
  • Datos técnicos: Información sobre el dispositivo, sistema operativo y navegador para optimizar la experiencia.

3. Uso de la Información

  • Brindar acceso a la demo y gestionar la solicitud de licencia.
  • Ofrecer soporte técnico y atención al cliente.
  • Mejorar el software y los servicios ofrecidos.
  • Enviar comunicaciones relacionadas con el producto, solo si el usuario lo autoriza.

4. Compartir Información

No compartimos tu información personal con terceros, salvo que sea necesario para cumplir con obligaciones legales o para la prestación del servicio (por ejemplo, proveedores de hosting para la versión en la nube).

5. Seguridad

Implementamos medidas de seguridad técnicas y organizativas para proteger tus datos contra accesos no autorizados, pérdida o alteración.

6. Derechos del Usuario

Puedes acceder, rectificar o eliminar tus datos personales en cualquier momento, contactándonos a través de contact@ademasistemas.com.

7. Uso de Cookies

El sitio web puede utilizar cookies para mejorar la experiencia de navegación. Puedes configurar tu navegador para rechazar cookies, aunque esto podría afectar el funcionamiento de algunas funciones.

8. Cambios en la Política

Nos reservamos el derecho de modificar esta política para adaptarla a novedades legislativas o mejoras del software. Se recomienda revisarla periódicamente.

9. Contacto

Para cualquier consulta sobre privacidad, puedes escribirnos a contact@ademasistemas.com.
